# JS的输入和输出

## 代码实现

```
<!DOCTYPE html>

<head>
    <script>
    
        // 输入框
        prompt('请输入您的年龄')

        // 弹出警示框,展示给用户的
        alert('计算的结果是')

        // 控制台输出，给程序员测试用的
        console.log('我是程序员，能看到')
        
    </script>
</head>
<body>
    
</body>
</html>
```
