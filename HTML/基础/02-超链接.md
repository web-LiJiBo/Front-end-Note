# 超链接

## 1.细节要点
   **（1）不进行跳转   `<a href='javascript:;'>删除</a>`**
   
   **（2）a表签的基本使用` <a href="跳转目标" target="目标窗口的弹出方式（_self【自身】  _blank【下一个】）">文本或图片</a>`**
